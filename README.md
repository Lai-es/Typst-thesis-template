
# Typst thesis templates for bachelor and master thesis

This template features two unique functions:
- A custom caption function for better readability of the table of contents (ToC)
- A custom function for subfigures

## Custom caption function

  <typst> 
    #caption[short][long]
  </typst>  
will show only the short caption in the ToC, while displaying the concatonated [short. long] caption under the figure.
